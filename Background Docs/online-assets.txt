ConverseTek — editor for BattleTech sim game conversations (.bytes)
  Project / source & releases:
    https://github.com/CWolfs/ConverseTek
    https://github.com/CWolfs/ConverseTek/releases
  Notes (high level; match releases to your game build):
    - Copy ShadowrunDTO.dll and ShadowrunSerializer.dll from your install:
        ...\BATTLETECH\BattleTech_Data\Managed\
      into the ConverseTek folder before running ConverseTek.exe.
    - Install the .NET runtime version requested by the release you download.
    - Export graphs into your mod folder: simGameConversations\*.bytes
    - Wire the exported conversation hex id into milestone System_StartConversation
      "value" (see Flashpoint Creation Guide §3.4 and §3.2).
